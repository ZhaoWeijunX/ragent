模板版本：V1.2

适用级别：P0 / P1 / P2 / P3（请在对应章节选用）
================================================================================

【使用说明】
1. 发现安全事件后，发现人须在 15 分钟内完成「初报」，并电话通知 SOC 值班电话：400-888-0601。
2. P0/P1 事件须同步通知 CISO 与相关业务 VP；是否对外通报由信息安全委员会决定。
3. 本模板字段不得删减；暂无信息填「待确认」，不得留空。
4. 发送范围：初报仅限应急群；终报按事件级别扩大，L4 数据相关须限制在知悉必要范围内。

五、监管报告要点（涉及个人信息或等保三级系统时参考）
================================================================================

向属地网信办/公安机关报告时，至少包含：

1. 事件发生的时间、地点、经过；
2. 涉及的个人信息种类、数量、可能被侵害的对象；
3. 已采取的措施；
4. 联系方式（CISO / 法务负责人）。

报告时限：
  - 个人信息安全事件：知悉后 72 小时内；
  - 等保三级以上系统：按属地公安网安部门要求（通常 1 小时内电话报告，24 小时内书面）。
